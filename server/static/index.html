Paper-Tracker Downloads

All Paper-Tracker downloads are hosted on Github Releases. You can download all necessary files from the "Assets" section for the latest release (the download buttons below are links to the files in the latest release). This way, it's easy to get the latest stable version of the App, the Server and the Tracker Firmware.

App

Most newer, more powerful Android phones should use the ARM64 download below. Older phones, as well as entry-level phones will probably use ARMv7. Some of the more powerful Android tablets use x86.

ARM64 ARMv7 x86

Unfortunately, iOS builds of the App cannot be provided due to missing infrastructure. However, it should be easy to build the App from source.
